# Express + MongoDB + hadlebars
## Description

A simple boilerplate template for any Express app, featuring mongoose for database connection and handlebars as the template engine.

## Folder structure
```
public/
  favicon.svg
  imgs
src/
  config/
    db.config.js
  routes/
    home.router.js
  server/
    server.js
  views/
    partials/
      header.handlebars
      footer.handlebars
    layouts/
      main.handlebars
  app.js
  index.html
  ...
```
